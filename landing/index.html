The problem

Voice typing is broken for devs

Cloud-based dictation is slow, inaccurate with technical terms, and sends your code context to someone else's server. macOS dictation can't spell useState. Siri thinks JWT is a radio station.

Your voice is 3x faster than typing, but the tools don't respect the way developers actually work.

The fix

Local. Fast. Developer-aware.

VoiceBar runs whisper.cpp on-device — no cloud, no latency, no privacy concerns. It knows technical vocabulary because the model runs locally with your full context.

Hebrew and English in the same sentence. Push-to-talk with F6. Text appears at your cursor. Works with Claude Code, Cursor, and any MCP client.

Capabilities

Built for how developers actually work

VoiceBar — the menu bar app
F6 push-to-talk Hold to record, release to transcribe and paste at cursor
whisper.cpp Local large-v3-turbo model — no cloud, no API key, no latency
Hebrew + English Bilingual transcription in the same sentence
Live waveform Floating pill with real-time audio visualization and teleprompter
Command mode Prefix with "command" to trigger actions instead of text input
Clip markers Mark important moments during recording for later reference
MCP server — give your AI a voice
voice_speak Non-blocking TTS with 20-output replay buffer and toggle control
voice_ask Blocking record + transcribe for conversational voice input
Voice modes announce, brief, consult, converse, think — match the interaction
Daemon lifecycle LaunchAgent singleton — auto-start, auto-restart, zero config
