# Pill Menu Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Add a native right-click pill menu for VoiceBar with snooze, microphone device selection, paste-last-transcript, and quit actions.

**Architecture:** Replace the current SwiftUI-only pill context menu with an app-owned `NSMenu` shown from the floating pill panel on right-click. Keep business logic in small helpers on `VoiceState`/`VoiceBarApp`, and keep device enumeration/menu wiring separate from the view so it can be unit-tested.

**Tech Stack:** Swift, AppKit, AVFoundation, SwiftUI, XCTest
